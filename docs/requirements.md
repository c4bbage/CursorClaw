# Cursor Claw 需求文档

## 项目概述

基于 Cursor CLI 和 OpenClaw 架构实现的自主 AI Agent 系统，支持实时通信、任务管理、并行执行和持久化记忆。

## 核心需求

### 1. 实时通信
- WebSocket 服务器支持多客户端连接
- 消息广播能力
- 双向实时数据流

### 2. 任务管理
- 创建、更新、查询任务
- 任务状态跟踪（pending/running/done）
- 用户级任务隔离

### 3. 并行 Agent
- 动态 spawn 子 Agent
- 独立会话上下文
- 异步任务执行

### 4. 记忆系统
- JSONL 格式持久化
- 用户级记忆隔离
- 上下文召回（最近 5 条）
- 会话管理（保留 10 轮）

### 5. 飞书集成
- 接收飞书消息
- 自动回复
- 事件驱动架构

## 工具能力

1. `execute_command` - 执行 Shell 命令
2. `create_task` - 创建任务
3. `spawn_agent` - 生成并行 Agent

## 使用场景

- 通过飞书发送任务指令
- Agent 自动创建任务并分配
- 并行执行多个子任务
- 实时推送执行状态
- 记忆历史对话上下文
